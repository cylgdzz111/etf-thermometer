# 产品需求文档（PRD）

## 产品定位

面向 A 股 / 港股 / 美股 ETF 投资者的估值温度计工具。
核心价值：用 PE / PB 历史分位数将当前估值量化为 0-100° 的"温度"，帮助用户判断市场冷热。

数据每日 19:00 ~ 21:00 更新。

---

## 功能模块

### 1. 市场估值概览（Dashboard）— Phase 3

**目标**：一眼看懂当前市场整体温度。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 全市场温度仪 | 综合 PE/PB 分位的 0-100° 温度值 + 文字描述（极度低估~极度高估） | P0 |
| 市场 PE 历史走势图 | 多条线（全市场/主板/创业板等），近 20 年，纯 SVG | P0 |
| 宽基指数卡片 | 各主要宽基的 PE、PB 值及分位（如全市场/主板/创业板/科创板/北交所） | P0 |
| 行业热力图 | 各行业当前估值分位，颜色热度，支持切换分位/涨跌/PE 三种模式 | P1 |
| 自选关注列表 | 快速浏览用户收藏的指数，点击跳转详情 | P1 |
| 市场切换 | A股 / 港股 / 美股，切换后所有数据联动刷新 | P0 |

---

### 2. 指数估值列表（Index List）— Phase 3

**目标**：在 300+ 指数中快速定位当前低估机会。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 指数表格 | 代码、名称、收盘价、涨跌幅、52周涨跌、距历史最高、PE、PE分位、PB、PB分位、温度 | P0 |
| 近 60 日迷你图 | 每行附带 sparkline 走势图 | P1 |
| 估值分段筛选 | 低估（PE分位<30）/ 正常 / 高估（PE分位>70） | P0 |
| 行业筛选 | 按行业板块筛选 | P0 |
| 搜索 | 按名称/代码模糊搜索 | P0 |
| 多列排序 | 点击列头排序，支持升/降序 | P1 |
| 我的收藏 | 收藏/取消，收藏列表优先展示 | P1 |
| 点击进入详情 | 整行可点击 | P0 |

---

### 3. 指数详情（Index Detail）— Phase 3

**目标**：深度了解单个指数的历史估值水位。

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 基本信息头 | 指数名称、代码、市场、今日收盘、涨跌幅、累计收益、距前高 | P0 |
| 综合温度 Banner | 温度值 + 温度条 + PE/PB 分位文字说明 | P0 |
| 价格 + PE 历史走势图 | 双 Y 轴，时间范围：近1年/近5年/上市以来 | P0 |
| PE 估值分布柱状图 | 历史 PE 频率分布，当前 PE 高亮，20/50/80 分位参考线 | P0 |
| PB 估值分布柱状图 | 同上 | P1 |
| 同板块指数推荐 | 同 sector 其他指数，便于横向对比 | P1 |
| 收藏 / 设为基准按钮 | 收藏操作；"设为基准"用于相关性页面（Phase 5） | P2 |
| 网格策略入口 | 跳转至网格策略页（Phase 5） | P2 |

---

### 4. 网格策略（Grid Strategy）— Phase 5

根据指数历史波动，自动计算网格买卖参数（间距、档位、资金分配）。

---

### 5. 相关性分析（Correlation）— Phase 5

多指数相关性矩阵，识别分散化配置机会。

---

## 数据需求

### 必须字段（每日更新）

| 字段 | 说明 |
|------|------|
| 收盘价 | 每日收盘 |
| PE（TTM） | 滚动 12 个月市盈率 |
| PB | 市净率 |
| PE 历史分位 | 基于近 10 年日数据计算 |
| PB 历史分位 | 同上 |

### 覆盖范围（目标）

| 市场 | 目标数量 | Phase |
|------|----------|-------|
| A股宽基 + 行业指数 | 200+ | Phase 2~3 |
| 港股主要指数 | 60+ | Phase 4 |
| 美股主要指数 | 90+ | Phase 4 |

---

## 非功能需求

| 项目 | 要求 |
|------|------|
| 首屏加载 | < 2s（静态资源 CDN 或本地 nginx） |
| API 响应 | < 300ms（列表接口） |
| 数据时效 | 每日 19:00 ~ 21:00 完成更新 |
| 移动端 | 暂不做专门适配，响应式布局保证基本可用 |
| 用户认证 | 暂不做登录，收藏存 localStorage |
